# 图书馆大数据可视化分析系统 python1195

## 项目概述

图书馆大数据可视化分析系统是基于Python开发的一套高效数据分析工具，旨在为图书馆提供数据驱动的决策支持。

## 技术栈

- 后端：Python
- 数据库：MySQL
- 前端：Web技术栈（未详细说明）
- 开发工具：PyCharm
- Python版本：3.7

## 功能模块

### 前台功能

- 首页
- 图书列表
- 数据可视化
- 关于

### 后台功能

- 首页
- 权限认证
- 图书管理
- 借阅管理
- 学生管理

## 系统角色

系统主要分为两类用户角色：普通用户和管理员。普通用户可通过前台功能进行图书查询和借阅；管理员则通过后台进行图书和学生信息管理。

## 运行环境

- 操作系统：兼容Windows、Linux、macOS
- Python环境：3.7版本及其以上
